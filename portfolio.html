Bloco de notas moderno

Este projeto consiste em aplicar meus conhecimentos em C#, programação orientada a objetos e WPF(Windows Presentation Foundation). O principal objetivo desse projeto é construir um bloco de notas moderno, visando a manutenibilidade de código e uma interface de usuário moderna e intuitiva, usando o .NET 9 para fins aprendizagem e aperfeiçoamento profissional.

O C# é uma linguagem de programação desenvolvida Microsoft, que faz parte da plataforma .NET. Ele usa o paradigma de programação orientado a objetos e foi influênciado por linguagens de programação como Object Pascal e principal Java.(Fonte: Wikipédia)

O .NET é uma plataforma de desenvolvimento gratuita multiplataforma e de código aberto para criar muitos tipos de aplicativos. Ele pode pexecutar programas escritos em vários idiomas, com o C# sendo o mais popular.(Fonte: Microsoft)

A versão mais recente da plataforma, o .NET 9.0, inclui novidades como o suporte à temas claro e escuro no WPF, conforme a imagem acima do projeto. O WPF (Windows Presentation Foundation) é uma interface de usuário para criar aplicativos para área de trabalho. A plataforma de desenvolvimento do WPF dá suporte a um amplo conjunto de recursos de desenvolvimento de aplicativos, incluindo um modelo de aplicativo, recursos, controles, gráficos, layout, associação de dados, documentos e segurança. (Fonte: Microsoft)

Com esse projeto acredido que poderei melhorar minhas habilidades de programação, adquirir mais conhecimentos sobre a plataforma .NET, c# e WPF, e ainda criar um aplicativo com funcionalidades incríveis.
