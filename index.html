Sobre mim

Meu nome é Gustavo, sou estudante de Engenharia de Software, gosto de videogames, ir ao cinema, e de ler bons livros, e estou sempre buscando novas maneiras de apreender programação de um jeito mais ágil, fácil e divertido.

Tenho bastante interesse no framework .NET, um ambiente de desenvolvimento de software criado pela Microsoft que visa unificar a criação de aplicações em uma única plataforma para que possa ser executada em qualquer dispositivo. Atualmente estou escrevendo um bloco de notas usando a linguagem de programação C#, uma liguagem de programação orientada a objetos com sintaxe bastante similar ao Java, e usando o Windows Presentation Foundation (WPF) como interface gráfica.

Falando sobre linguagens de programação, também tenho interesse em aprofundar meus conhecimentos em Java, JavaScript, Python, Rust e C++. Além disso estou sempre disposto a encontrar soluções para problemas existentes, mesmo aqueles que ainda não existam uma solução.

Ao contrário do que muitas pessoas pensam, programar não é somente escrever um monte de código, é na verdade construir soluções profissionais e criativas com base em boas práticas de programação e nos padrões de arquitetura de software e de engenharia de software.

De acordo com Robert C. Martin, "bons sistemas de software começam com um código limpo. Por outro lado, se os tijolos não são bem feitos, a arquitetura da construção perde a importância" (Arquitetura Limpa, p.57).

Através dessa analogia posso concluir que existem várias maneiras programar, mas se não fizermos de uma maneira eficaz, o código não funcionará como esperado.
